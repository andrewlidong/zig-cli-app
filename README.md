# babyline

a small zig CLI demo. originally built by following [this guide](https://rebuild-x.github.io/docs/#/./zig/terminal/cli?id=step-6-testing-your-cli) and extended from there.

## build

```
zig build           # binary -> ./zig-out/bin/babyline
zig build test      # run the test suite
zig build docs      # regenerate docs/ and the CLI reference below
```
